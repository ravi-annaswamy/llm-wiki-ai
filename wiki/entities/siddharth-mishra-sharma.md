# Siddharth Mishra-Sharma

> **Anthropic Discovery team researcher (cosmology background); author of the CLAX long-running-agentic-coding walkthrough.** First named practitioner catalogued in the wiki for the long-horizon autonomous coding pattern applied to scientific computing.

His April 2026 Anthropic post walks through building [clax-project](clax-project.md) — a differentiable JAX Boltzmann solver — using Claude Opus 4.6, benchmarked against the [class-boltzmann-solver](class-boltzmann-solver.md) C reference. He is explicit that cosmology is adjacent to, not inside, his core expertise — part of the point, since the experiment tests whether a non-expert can steer an agent further than domain experts have gone solo. Following the agent's commit history turned out to be an effective way to absorb the science.

The post is the reference document for several wiki concept pages: [long-running-agentic-coding](../concepts/long-running-agentic-coding.md), [test-oracle-for-agents](../concepts/test-oracle-for-agents.md), [agent-persistent-memory](../concepts/agent-persistent-memory.md), [ralph-loop](../concepts/ralph-loop.md), [agent-driven-scientific-computing](../concepts/agent-driven-scientific-computing.md).
